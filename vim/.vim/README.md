# Vim config

```
├── autoload
├── bundle                  # Populated by VimPlug
├── colors
├── compiler
├── config                  # Plugins to install
├── ftdetect                # Commands to set syntax based on file exts
├── ftplugin                # Configuration of lang specific plugins
├── gvimrc                  # Gvim specific configuration
├── haskellmode.config      # Haskell tip
├── plugin                  # Global vim configurations
├── snippets
├── spell
├── syntax
├── UltiSnips
└── vimrc                   # Symlink to ~/.vimrc
```
